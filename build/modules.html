drawDgm(boundary=None, epsilon=0.5, color=None)¶

Plots persistence diagram.

Parameters

  • boundary – The diagram is drawn on [0,boundary]x[0,boundary].

  • epsilon – If boundary not given, then it is determined to be the max death time from the input diagram plus epsilon.

  • color – Color for plotting points in diagram.
